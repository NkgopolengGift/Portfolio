InfluencerConnect

InfluencerConnect is a platform designed to connect influencers with potential sponsors by aggregating key statistics from various social media platforms like YouTube, TikTok, Facebook, etc. As part of a school project, I developed the entire backend using Django and PostgreSQL, handling user authentication, data aggregation, and seamless sponsor-influencer interaction. The platform helps brands identify influencers based on social media performance metrics using HTML, CSS, and JavaScript for the front-end.

JobMatchup

JobMatchup is a platform aimed at helping individuals without formal qualifications find jobs such as cleaner, gardener, and more. I built the application using Spring Boot (Java) for the backend, PostgreSQL for the database, and JavaScript, HTML, and CSS for the front-end interface. The platform helps reduce unemployment by providing accessible job listings to users in South Africa.
